About JuubiAI

JuubiAI is a research prototype exploring intelligent query routing across specialised AI models. The core idea: no single model excels at everything. By decomposing complex queries and routing each piece to the best-suited model, we obtain higher quality answers than any monolithic approach.

How it works

01

Query Decomposition

Complex queries are decomposed into independent reasoning subtasks.

02

Graph Construction

Dependencies become a directed acyclic graph enabling parallel execution.

03

Domain Routing

Each task is classified into Climate, Medical, Legal, Code and more.

04

Model Retrieval

Models are retrieved from a capability index according to domain.

05

Parallel Execution

Selected models execute concurrently while streaming partial results.

06

Aggregation

Outputs are synthesised into one coherent final answer.
