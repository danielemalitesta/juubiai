About JuubiAI

JuubiAI is a research prototype exploring intelligent query routing across specialised AI models. The core idea: no single model excels at everything. By decomposing complex queries and routing each piece to the best-suited model, we can achieve higher quality answers than any monolithic approach.

How it works

01

Query Decomposition

Complex queries are broken down into independent, parallel subtasks — each representing a distinct reasoning unit.

02

Graph Construction

Dependencies between subtasks are mapped into a directed acyclic graph, enabling parallel execution.

03

Domain Routing

Each subtask is classified into a semantic domain — Climate, Code, Medical, Legal, and more.

04

Model Retrieval

Candidate models are retrieved and scored for each domain using a model capability index.

05

Parallel Execution

Subtasks execute concurrently on selected models, streaming partial results.

06

Aggregation

Results from all subtasks are synthesised into a single coherent final answer.
